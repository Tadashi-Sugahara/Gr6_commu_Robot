スクリプトを使用するためには下記のコマンドを実行してください。
  ```
  cd ~/ai2024/Group6_AIalarm
  python -m venv venv
  source venv/bin/activate
  （必要に応じてライブラリをインストール）
  python main.py 2>/dev/null
  ```
初回、またはライブラリが追加された場合、下記の通りインストールする必要があります。  
  ```
  sudo apt-get update
  sudo apt-get install flac
  pip install -r requirement.txt
  ```
